Вы раньше играли у нас в Пейнтбол?

Пейнтбол - это командная игра, компания игроков делится на две команды, цель каждой команды поразить участников другой команды из маркера – оружия стреляющего шариками с краской. После попадания шарика в игрока игрок «выбывает» из игры.

Есть много сценариев игры, сценарий это миссия которую нужно выполнить, наиболее распространенные:

Стенка на стенку – команды стартуют с разных концов площадки, цель поразить участников другой команды.

Захват крепости – каждая команда начинает игру в своей крепости, задача захватить крепость другой команды.

Захват флага – цель игры – захватить флаг чужой команды и принести флаг в свою базу.

Кроме этого есть сценарии с участием пиротехники, ракет, взрывов. Об этом подробно расскажет наш инструктор, который будет проводить игру.
За время игры, компания игроков проходит 5-6 разных миссий.
